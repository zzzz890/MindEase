Methodology

How the ML Model Works

A pipeline from raw text to emotional intelligence — built on real-world social data.

01
📥
Data Collection
3,534 labeled tweets collected from 195 countries, across 6 age groups and 3 time windows — morning, noon, and night.
02
🔤
Text Preprocessing
Tokenization, stopword removal, lowercasing, and URL stripping clean the raw text before feature extraction.
03
🧮
Feature Engineering
TF-IDF vectorization combined with contextual embeddings captures both term frequency and semantic meaning.
04
🤖
Sentiment Classification
A Naive Bayes + logistic regression ensemble classifies inputs as positive, negative, or neutral with confidence scores.
05
💬
Response Generation
Based on detected sentiment, the chatbot selects empathetic, context-aware responses and coping suggestions.
06
📊
Continuous Learning
User feedback loops improve accuracy over time, with special attention to demographic and temporal patterns.
